# 廃棄層の発掘家 v17.5.10

## v17.5.10 廃棄層入口の遷移方式修正

- 中央タップで廃棄層へ入る方式を廃止。
- 入口マス、入口前、入口周辺から入口方向へ移動入力すると廃棄層へ入るように修正。
- 入口付近のラベルとステータス文を「入口へ進む」に変更。
- v17.5.7/v17.5.8で発生した、入口に立っても中央タップで遷移しない問題を回避。
- 既存の9分割タップ移動、会話、通行可否の明度表現、スマホ拡大抑止は維持。


## v17.5.6 会話導線の改善

- 住人の会話を説明文ではなく、各キャラクターの短い発言に調整。
- 初回会話で「東の隔壁から廃棄層へ入る」「本命は浄水コア」「危なくなる前に戻る」を理解できるように修正。
- 水守り、老発掘家、修理屋、見張り、記録係の役割を明確化。
- 発掘回数、最深到達、浄水コア回収状況に応じて会話内容が変わるように調整。
- スマホ画面で読み切れるよう、会話表示は短文3行に整理。

## v17.5.5 通行可否の視認性改善

- 通れないマスを薄いブロックで覆う表現をやめ、背景グラフィックの上から暗めの明度補正で表現。
- 通れるマスは背景を少し明るくし、進める場所を直感的に判別しやすくした。
- 集落・廃棄層の両方で、通行可能/不可能の見分けを明度差中心に調整。
- v17.5.4 のスマホ拡大抑止、v17.5.3 の初期画像改善、9分割タップ移動は維持。

## v17.5.5 スマホのダブルタップ拡大抑止

- 画面2回タップ時のブラウザ拡大を抑止
- ピンチ拡大/gesture系イベントをゲーム画面では抑止
- タップ操作領域の touch-action を強化
- 長押し選択/タップハイライトを抑止
- v17.5.3 のマップ初期表示修正と9分割タップ操作は維持

## v17.5.5 画像・マップ初期表示のデバッグ修正

- 画像アセットにも `?v=17.5.5` を付与し、古いPNGキャッシュを読まないように修正。
- CSS背景画像にも `?v=17.5.5` を付与。
- 画像読み込み中に旧Canvas描画へフォールバックする4秒タイムアウトを廃止。
- 読み込み完了までローディング表示にし、初期表示でチープな仮描画を見せないように変更。
- 画像ロード進行時に即時再描画するイベントを追加。
- 集落開始時のメニューオーバーレイを廃止し、初期状態からマップ上を直接歩けるように修正。

## v17.5.5 9分割タップ移動

- 画面を見えない9分割エリアとして扱い、タップ位置で8方向移動。
- 方向パッドの矢印UIを廃止。
- NPCのいる方向へ移動入力すると自動で会話。
- 敵のいる方向へ移動入力すると自動で攻撃。
- 中央タップは足踏み/隣接確認。足元に遺物があれば拾う。廃棄層入口へは入口方向への移動で入る。
