# 廃棄層の発掘家 v19.1.0

核戦争後の世界。AIロボットが作り、捨て続ける「廃棄層」に潜り、
集落の浄水機関を動かす燃料「アクアコア」(地下30階)を持ち帰る
シレン/トルネコ系のターン制ローグライク。

## v19.1.0 の変更点(操作感・見た目の調整)

- **スマホ操作の修正**: 長押しで一気に動きすぎる問題を解消。
  - 1回タップ = 確実に1歩だけ
  - 長押し = 約0.32秒待ってから、1歩ずつ間隔(約0.17秒)を空けて連続移動
  - 2回連続タップ = ダッシュ(従来通り)
- **立ち絵を小さく**: 高さをマスの1.7倍→**1.25倍**に縮小。密集しても重なりにくく、視界を圧迫しないように。
- **集落を完全タイル化**: 固定背景画像(base.jpg)を廃止し、家・屋根・扉・浄水機・涸れ井戸・木箱・焚き火・道・柵をすべてタイルパーツで構築。中央広場の焚き火は炎がゆらめきます。
- タイトル背景(title.jpg)は引き続き使用。

## v19.0.0 の変更点(画像アセット導入)

- キャラ・敵・NPCを立ち絵として描画(足元に影、奥行きソート)
- 歩行は上下バウンド+影の伸縮+踏み込みモーション
- ダンジョンの床・壁・特殊オブジェクトをイラストタイルで描画
- 画像が読めない場合はコード生成スプライトに自動フォールバック
- 元22MBの画像を約0.6MBに圧縮(集落・ダンジョン背景は不要になり削除)

## ファイル構成

```
index.html / style.css
js/   … rng, config, assets, sprites, audio, dungeon, ui, input, game
assets/
  chars/    … 立ち絵(主人公・NPC・敵に流用)
  tiles/    … 床・壁・特殊オブジェクト・装飾
  items/    … 種別アイコン
  bg/       … タイトル・集落・ダンジョン背景
  fx/       … エフェクト
```

旧 v17 の `src/` は不要です(削除して構いません)。

## 操作

| 操作 | スマホ | PC |
|---|---|---|
| 移動(8方向) | 行きたい方向をタップ | 矢印 / hjkl+yubn |
| 連続移動 | 長押し | キー押しっぱなし |
| ダッシュ | 同方向を素早く2回タップ | Shift+方向 |
| 攻撃 | 敵の方向をタップ | 敵の方向へ移動 |
| 階段・拾う | 自分をタップ / 「足元」 | Enter / Space |
| 道具 | 「道具」ボタン | i |
| 足踏み | 「待機」ボタン | . または s |

## ルール(シレン系)

- 自分が1歩動くと、敵も1歩動くターン制
- 床の道具は歩くだけで自動で拾う
- 満腹度が尽きるとHPが減り続ける
- 倒れると**道具を全て失い、レベルも1に戻る**(集落へ送還)
- 「帰還タグ」で道具を持ったまま帰還できる
- 装備には強化値(+1〜)があり、腐食粘体や錆噴霧で下がる

## セキュリティ・通信について

外部通信は**一切行いません**。

- CSP: `default-src 'none'; script-src 'self'; style-src 'self'; img-src 'self' data: blob:`
  (画像はすべて同一オリジンの `assets/` 配下のローカルファイル)
- fetch / XHR / WebSocket / 外部script / iframe / serviceWorker: 不使用
- 音: すべてWebAudioでコード生成

### localStorage の使用内容

| キー | 内容 |
|---|---|
| `hakkutsuka_v18_save` | セーブ(レベル・HP・持ち物・装備・クリア回数) |
| `hakkutsuka_v18_record` | 発掘記録(最深到達階・出発回数・コア回収回数) |

### 通信確認手順

1. 開発者ツール → Network タブを開く
2. 再読み込みしてプレイ
3. `index.html` / `style.css` / `js/*` / `assets/*` 以外の
   リクエストが発生しないことを確認(CSP違反はConsoleに出ます)
