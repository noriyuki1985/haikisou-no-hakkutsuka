# 廃棄層の発掘家 v17.5.24

## v17.5.24 一本道ダンジョン・生成バランス再調整

画像生成なし。画像編集なし。コード側の生成バランスと探索テンポを調整。

### 変更内容

- ダンジョン進行を「一本道・下層へ進むのみ」の前提に寄せた。
- リフト/コア配置を距離ベースではなく、部屋順ベースで「ほどよく前方」に配置。
- 部屋数を 10 から 8 に圧縮し、リフト到達までの体感距離を短縮。
- 敵数、アイテム数、罠数、汚染床数を再調整。
- 特殊部屋の出現数に上限を追加し、当たり外れの振れ幅を抑制。
- 廃材部屋、端末部屋、汚染部屋の追加敵/追加アイテム/追加汚染を弱めた。
- 汚染床の全体上限を下げ、序盤の圧迫感を抑制。
- 移動アニメーションをさらに短縮し、スマホ操作時のテンポを改善。
- 出発地点へ戻っても通常移動では帰還しないように変更。撤退は帰還タグまたは途中帰還操作を使う。
- ヘルプ、探索メニュー、ラベル文言を「戻れない/リフトへ進む」前提に調整。

### 確認観点

- リフト到達までが長すぎない。
- 序盤の敵が多すぎない。
- アイテムが多すぎず、足りなさすぎない。
- 汚染床が邪魔すぎない。
- 部屋タイプごとの当たり外れが極端でない。
- 出発地点へ戻っても通常移動では帰還しない。
- 帰還タグまたは途中帰還操作で撤退できる。
