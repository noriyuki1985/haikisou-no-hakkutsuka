## 操作

### スマホ（タッチ）

```txt
方向パッド: 8方向移動（中央=足踏み）
拾う/使う: 足元に遺物があれば拾う、なければ選択中の所持品を使用
所持品: 一覧を開き、タップで選択。使う/投げる/置く/閉じる
投げる: 選択中の所持品を投げる
メニュー: 探索メニュー（投げる/調べる/帰還/記録/ヘルプ/閉じる）
ストーリー/会話/結果画面: 画面内のボタンで送る/閉じる/新規発掘
```

### PC（キーボード）

```txt
Enter: 探索開始 / ストーリー送り / 会話を閉じる
B: 拠点
S: タイトル/拠点でストーリー表示、探索中は下移動
E: タイトル/拠点でエンディング再読、探索中は斜め移動
L: タイトル/拠点で発掘記録
P: 探索中に途中帰還
H: ヘルプ
Esc: 閉じる
1〜5: 拠点で住人と会話
矢印/WASD: 移動
Q/E/Z/C またはテンキー: 斜め移動
.: 足踏み
G: 拾う
I: 所持品一覧
1〜8: 探索中に所持品を選択
U: 選択中の所持品を使用
[ ]: 所持品選択
X: 置く
T: 投げる
F: 周囲を調べる
M: 探索中は探索メニュー、タイトル/拠点では識別設定切替
N: 新規発掘
Shift+O: 保存データ初期化
```

## 世界観

核戦争後、人類は高度な科学を失いました。しかし、旧時代の自律AIは今も止まらず、建物や部品を作っては壊し、廃棄し続けています。人間はAIが捨てたものを拾い、直し、使い回すことで集落を維持しています。

主人公は発掘家です。集落の浄水装置を延命するため、再構成され続ける廃棄区域へ潜り、浄水コアを探します。

## GitHub Pages配置

リポジトリ直下は以下の構成にしてください。

```txt
index.html
style.css
README.md
src/
assets/
```

`src/` と `assets/` の階層を崩すと動きません。
