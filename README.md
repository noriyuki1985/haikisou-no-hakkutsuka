# 廃棄層の発掘家 v20.0.0

核戦争後の世界。AIロボットが作り、捨て続ける「廃棄層」に潜り、
集落の浄水機関を動かす燃料「アクアコア」(地下30階)を持ち帰る
シレン/トルネコ系のターン制ローグライク。

## v20.0.0 の変更点(視認性・操作・演出の強化)

- **ダッシュは3回タップに変更**: 誤発動を防止。素早く3回タップで突進。
  (2回タップでは発動しません)。突進速度も少し落として見やすく。
- **長押し中に指をスライドして方向転換**: 押したまま指を動かすと進行方向が変わります。
- **タップのデッドゾーンを調整**: 自分の近くをタップ=足元(階段/拾う)、外側=その方向へ移動。
- **全体を明るく**(視認優先度 ①UI ②キャラ/アイテム ③背景 の順):
  - UI(文字/メニュー/アイコン/ボタン)を最も高コントラストに
  - キャラ・アイテムを明度/彩度UP
  - 背景(床)を明るく、壁はやや沈めて障害物として区別
- **アイテムが光る**: 床のアイテムに種別色の発光と足元リング、上昇する光の粒。
  グラフィック(装飾)と拾える物が一目で区別できます。
- **インタラクト可能物を強調**: 階段/リフトは明滅する光と矢印(▼降りる/▲帰還)、
  アクアコア台座は床から強く発光。
- **壁の立体化(オートタイル)**: 壁を周囲の床配置から判定し、
  平壁/内角/外角/端を自動で描き分け(回転対応)。通路や部屋の輪郭が立体的に。

## ファイル構成

```
index.html / style.css
js/   … rng, config, assets, sprites, audio, dungeon, ui, input, game
assets/
  chars/    … 立ち絵(主人公・NPC・敵に流用)
  tiles/    … 床・壁・特殊オブジェクト・装飾
  items/    … 種別アイコン
  bg/       … タイトル・集落・ダンジョン背景
  fx/       … エフェクト
```

旧 v17 の `src/` は不要です(削除して構いません)。

## 操作

| 操作 | スマホ | PC |
|---|---|---|
| 移動(8方向) | 行きたい方向をタップ | 矢印 / hjkl+yubn |
| 連続移動 | 長押し | キー押しっぱなし |
| ダッシュ | 同方向を素早く2回タップ | Shift+方向 |
| 攻撃 | 敵の方向をタップ | 敵の方向へ移動 |
| 階段・拾う | 自分をタップ / 「足元」 | Enter / Space |
| 道具 | 「道具」ボタン | i |
| 足踏み | 「待機」ボタン | . または s |

## ルール(シレン系)

- 自分が1歩動くと、敵も1歩動くターン制
- 床の道具は歩くだけで自動で拾う
- 満腹度が尽きるとHPが減り続ける
- 倒れると**道具を全て失い、レベルも1に戻る**(集落へ送還)
- 「帰還タグ」で道具を持ったまま帰還できる
- 装備には強化値(+1〜)があり、腐食粘体や錆噴霧で下がる

## セキュリティ・通信について

外部通信は**一切行いません**。

- CSP: `default-src 'none'; script-src 'self'; style-src 'self'; img-src 'self' data: blob:`
  (画像はすべて同一オリジンの `assets/` 配下のローカルファイル)
- fetch / XHR / WebSocket / 外部script / iframe / serviceWorker: 不使用
- 音: すべてWebAudioでコード生成

### localStorage の使用内容

| キー | 内容 |
|---|---|
| `hakkutsuka_v18_save` | セーブ(レベル・HP・持ち物・装備・クリア回数) |
| `hakkutsuka_v18_record` | 発掘記録(最深到達階・出発回数・コア回収回数) |

### 通信確認手順

1. 開発者ツール → Network タブを開く
2. 再読み込みしてプレイ
3. `index.html` / `style.css` / `js/*` / `assets/*` 以外の
   リクエストが発生しないことを確認(CSP違反はConsoleに出ます)
