# 廃棄層の発掘家 v17.5.27

## v17.5.27 戦闘テンポ・スマホ所持品バー調整

画像生成なし。画像編集なし。

### 変更内容

- 戦闘テンポを調整。
  - 攻撃演出時間を短縮。
  - FX表示時間を短縮。
  - 移動アニメーションを微短縮。
  - 画面揺れを少し弱め、連続操作時の視認性を改善。
- 攻撃ログを短縮。
  - 攻撃、被弾、撃破、射撃、砲撃、投擲ログを短い表記へ変更。
  - 敵の予兆ログも短縮。
  - スマホでログが画面を占有しにくいよう最大ログ行数を抑制。
- 敵接触時のフィードバックを調整。
  - 敵へぶつかって攻撃した瞬間に短い反応FXを追加。
  - 被弾・撃破ログを短くして、連続戦闘のテンポを改善。
- スマホ所持品バーを調整。
  - 所持品バーを1列構成に変更し、押し間違いを減らす。
  - 「使う」「投げる」「捨てる」ボタンを拡大。
  - 所持品スロットを横スクロールしやすく調整。
  - 8個所持時も選択中スロットが見えやすいよう、自動で中央寄せ。
  - 横スクロール時のタッチ挙動を改善。

### 確認観点

- 敵にぶつかった時、攻撃したことが分かる。
- 戦闘ログが短く、連続戦闘でも邪魔になりにくい。
- スマホで「使う」「投げる」「捨てる」が押しやすい。
- 所持品が8個ある時、横スクロールで全アイテムを選べる。
- 中央タップによる拾う/使う動作が壊れていない。
