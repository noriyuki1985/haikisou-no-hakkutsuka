# 廃棄層の発掘家 v21.5.0

核戦争後の世界。AIロボットが作り、捨て続ける「廃棄層」に潜り、
集落の浄水機関を動かす燃料「アクアコア」(地下30階)を持ち帰る
シレン/トルネコ系のターン制ローグライク。

## v21.0.0 の変更点

- **ダッシュを4回タップに変更 + 高速化**: 同方向に素早く4回タップで突進。
  突進の移動間隔を70msまで高速化(従来比約2倍速)。
- **ロボットステーション(モンスターハウス)を実装**: B3F以降、一定確率で
  「機械が部屋いっぱいに詰まった部屋」が出現(深いほど出やすい / 最大12体)。
  踏み込むと**警報**(画面フラッシュ+赤い縁の点滅+クラクション)と共に全機が一斉起動。
  代わりに**報酬アイテムも多め**に配置(ハイリスク・ハイリターン)。
  休眠チップ・EMP・帰還タグの価値が跳ね上がります。
- **ミニマップ強化**: 拡大して見やすく。アイテムを種別色のドット、
  階段/リフト/コアを点滅アイコン、敵を赤ドット(ボスは大きく)で表示。
  ロボットステーション起動中はミニマップの枠が赤く点滅。

## ファイル構成

```
index.html / style.css
js/   … rng, config, assets, sprites, audio, dungeon, ui, input, game
assets/
  chars/    … 立ち絵(主人公・NPC・敵に流用)
  enemies/  … 敵専用立ち絵(未配置時はcharsへフォールバック)
  tiles/    … 床・壁・特殊オブジェクト・装飾
  items/    … 種別アイコン
  bg/       … タイトル・集落・ダンジョン背景
  fx/       … エフェクト
```

旧 v17 の `src/` は不要です(削除して構いません)。

## 操作

| 操作 | スマホ | PC |
|---|---|---|
| 移動(8方向) | 行きたい方向をタップ | 矢印 / hjkl+yubn |
| 連続移動 | 長押し | キー押しっぱなし |
| ダッシュ | 同方向を素早く2回タップ | Shift+方向 |
| 攻撃 | 敵の方向をタップ | 敵の方向へ移動 |
| 階段・拾う | 自分をタップ / 「足元」 | Enter / Space |
| 道具 | 「道具」ボタン | i |
| 足踏み | 「待機」ボタン | . または s |

## ルール(シレン系)

- 自分が1歩動くと、敵も1歩動くターン制
- 床の道具は歩くだけで自動で拾う
- 満腹度が尽きるとHPが減り続ける
- 倒れると**道具を全て失い、レベルも1に戻る**(集落へ送還)
- 「帰還タグ」で道具を持ったまま帰還できる
- 装備には強化値(+1〜)があり、研磨機や錆噴霧で下がる

## セキュリティ・通信について

外部通信は**一切行いません**。

- CSP: `default-src 'none'; script-src 'self'; style-src 'self'; img-src 'self' data: blob:`
  (画像はすべて同一オリジンの `assets/` 配下のローカルファイル)
- fetch / XHR / WebSocket / 外部script / iframe / serviceWorker: 不使用
- 音: すべてWebAudioでコード生成

### localStorage の使用内容

| キー | 内容 |
|---|---|
| `hakkutsuka_v18_save` | セーブ(レベル・HP・持ち物・装備・クリア回数) |
| `hakkutsuka_v18_record` | 発掘記録(最深到達階・出発回数・コア回収回数) |

### 通信確認手順

1. 開発者ツール → Network タブを開く
2. 再読み込みしてプレイ
3. `index.html` / `style.css` / `js/*` / `assets/*` 以外の
   リクエストが発生しないことを確認(CSP違反はConsoleに出ます)
