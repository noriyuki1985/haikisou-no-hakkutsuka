## 起動方法
- `index.html` をブラウザで開いてください
- スマホ実機確認は GitHub Pages などの静的ホスティング想定です

## デプロイ用 git コマンド
```powershell

git add .
git commit -m "Update game to v22.8.1 refine village onboarding and add animated town NPCs"
git push origin main
```
