# 敵固有行動 実装メモ v21.4.1

## v21.2.0相当: 序盤〜中盤

- 回収ドローン: 床アイテムを拾い、所持中は逃走。撃破時に回収物を落とす。
- 吸引機: 同一直線・斜線上の主人公を1マス引き寄せる。
- 研磨機: 近接攻撃時に武器または防具の強化値を低下させる。
- 自爆セル: 隣接で起爆準備、2カウント後に爆発。主人公と敵を巻き込む。
- 修復ビット: HP割合が最も低い周囲の敵を優先修復する。

## v21.3.0相当: 中盤ギミック

- 警報ビーコン: 主人公を発見すると警報カウントを進め、増援を2体召喚する。
- 展開シールド: 周囲の敵に一時シールドを付与し、受けるダメージを下げる。
- 補給ポッド: 周囲の敵に攻撃・防御強化を付与する。
- 投棄機: 直線射撃で廃材を投げ、着弾付近に鉄くずを残す。
- 掘削機: 直線上の主人公へ突進し、進路の壁を破壊する。
